# Smart Music Player

> Music player with ML mood detection and auto-playlist generation

## Overview
This project is part of a comprehensive AI/ML portfolio covering python music.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/smart-music-player.git
cd smart-music-player
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: Python, Music, Machine Learning

## Usage
```bash
python main.py
```
Run with `--help` for full options.
